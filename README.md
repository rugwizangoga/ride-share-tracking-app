# 📖 Ride Share Tracking App <a name="about-project"></a>

**ride-share-tracking-app**  is a sophisticated web platform designed for drivers navigating fixed routes in Kigali, like the journey from Nyabugogo to Kimironko. Powered by the Google Maps API, this application provides a user-friendly interface that maps out the entire route and pinpoints designated stops. Offering real-time tracking of the driver's location and accurate estimated time of arrival (ETA) updates for each stop, Ride Share Tracking App ensures seamless coordination between drivers and passengers for a smooth and efficient travel experience in Kigali.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Git
- Shell
- Browser
- Code Editor

### Setup

Clone this repository to your desired folder:

- $ git clone https://github.com/rugwizangoga/ride-share-tracking-app.git

- $ cd ride-share-tracking-app 

### Install

Install this project with:

- $ npm install

### Usage

To run the project, execute the following command:

- npm start
